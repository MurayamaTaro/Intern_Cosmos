# Intern Cosmos


## データセット

### VRipt
- https://github.com/mutonix/Vript
- 解像度720×1280、フレーム数121に整形済み
  - 解像度はアスペクト比を保ちつつセンタークロップ&スケーリングで処理
  - フレーム数は長尺動画を前から重複なしで121フレームずつ切り出し
- 付属カテゴリ名でドメイン分割済み
- テキスト埋め込み済み

### UltraVideo
- https://github.com/xzc-zju/UltraVideo
- short_960をダウンロード
- 解像度720×1280、フレーム数121に整形済み
  - 解像度はアスペクト比を保ちつつセンタークロップ&スケーリングで処理
  - フレームが足りないときは補間で処理
- コーデック: H.264
- ドメイン分割はしていない
- テキスト埋め込みもしていない


## 注意
- conda使用はNG。ライセンスを取っていないため。
